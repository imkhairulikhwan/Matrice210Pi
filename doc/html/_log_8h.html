Detailed Description

This class provides log functions.

Version
1.0
Date
Jul 27 2018
Author
Jonathan Michel The goal is to use DJI::LOG methods to print log on console and send log to mobile application. Three types are available : LSTATUS, LERROR, LDEBUG User can use DJI::OSDK::Log class to enable/disable specified log type DSTATUS, DERROR, DDEBUG methods can always be used to log local messages on the console and not send them to mobile.
